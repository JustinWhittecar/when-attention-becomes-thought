# Papers

Local copies of the primary sources, organized loosely by chapter.

This folder is **gitignored**. Do not commit PDFs. Keep them here so they are one click away when you are writing, but keep them out of the repository history.

Suggested naming convention: `AuthorYear-short-title.pdf` (e.g. `Vaswani2017-attention-is-all-you-need.pdf`).
